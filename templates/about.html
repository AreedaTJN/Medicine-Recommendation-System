About the Health Care Center Project

Health Care Center is a web application project that develops a preliminary health information recommendation system using Artificial Intelligence (AI). It was created for the primary purpose of learning and developing technical skills and is not intended for real medical use.

Project Goals

This project was built to explore and apply the capabilities of Machine Learning to analyze user-inputted symptoms and present useful information, such as potential diseases, descriptions, precautions, and preliminary self-care advice. This is all part of studying and developing skills in:

  • Web Development: Creating and developing web applications with a Python framework.
  • Machine Learning: Deploying a trained Machine Learning model.
  • User Interface (UI): Designing a user-friendly and intuitive website interface.

Technologies Used

  • Backend: Python (Flask)
  • Machine Learning: Scikit-learn, Pandas, Numpy, Pickle
  • Frontend: HTML, CSS, Bootstrap 5

Important Warning & Disclaimer!

The information displayed in this web application is solely the result of processing by an AI model created for educational purposes. This information must not be used for self-diagnosis or as a substitute for professional medical advice under any circumstances.

If you are experiencing symptoms or feel unwell, please consult a doctor or the nearest medical facility for a proper diagnosis and treatment.
